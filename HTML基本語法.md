# HTML基本語法

## Hello World

```markup
<html>
<head>
    <title>Hello World 網站</title>
</head>
  
<body>
    <div>
        <p>第一個段落</p>
        <h1>第1層標題</h1>
        <h2>第2層標題</h2>
        <h3>第3層標題</h3>
        <h4>第4層標題</h4>
        <h5>第5層標題</h5>
        <h6>第6層標題</h6> 
    </div>
</body>
</html>
```

## 段落

```markup
<p>第一個段落</p>
```

## 區塊

```markup
<div>
    <p>第一個段落</p>
    <h1>第1層標題</h1>
    <h2>第2層標題</h2>
    <h3>第3層標題</h3>
    <h4>第4層標題</h4>
    <h5>第5層標題</h5>
    <h6>第6層標題</h6> 
</div>
```

## 強制換行

```markup
<p>前文<br>換行</p>
```

## 超連結

```markup
<a href="http://www.google.com">Google</a>
```

## 圖片

```markup
<img src="http://st2.ilvs.ilc.edu.tw/~s514119/img/GOOGLE1.jpg">
```

## 文字格式化

| 標籤 | 描述 |
| :---: | :---: |
|  &lt;b&gt; | 粗體 |
|  &lt;i&gt; | 斜體 |
|  &lt;sup&gt; | 上標 |
|  &lt;sub&gt; | 下標 |
